How this works

  • Pinyin & word meanings are looked up instantly against a bundled copy of CC-CEDICT (~120,000 entries) — entirely offline, no network call, no rate limit.
  • Sentence translations use the free MyMemory API — no key required, capped at 5,000 characters/day per visitor (50,000/day if you add your email in the code).
  • Speak it back — click any word to hear it pronounced via your browser's built-in voices.
  • Build a deck — export the vocabulary list as CSV and import it into Anki or Quizlet.
  • Segmentation uses a longest-match algorithm against the dictionary, so very rare words or names it doesn't recognise may show as single characters — that's expected for a lightweight offline dictionary.
