How this works

  • Pinyin, meanings, and HSK levels are looked up instantly against bundled offline datasets (CC-CEDICT + the complete HSK 3.0 vocabulary list) — no network call, no rate limit.
  • Sentence translations use the free MyMemory API — no key required, capped at 5,000 characters/day per visitor (50,000/day if you add your email in the code).
  • Character decomposition (which components a character is built from) comes from the open-source Make Me a Hanzi project.
  • Writing practice, single character or whole sentence, uses HanziWriter for stroke order and tracing.
  • Review mode uses a simple spaced-repetition schedule: words and sentences you grade "Good" or "Easy" come back further apart; "Again" brings them back sooner.
  • Everything you save lives in this browser's local storage — it won't follow you to another device unless you use the export/import backup buttons.
